# アプリケーション情報

概要:

```
Vantan Dollar(VTD)を管理するためのサイト

API:
  使い方:`from VTDbank import api`みたいな感じでインポートして使用

  関数:
    CreateClaim:アプリケーションからの請求を作成します。
    AddVTD:アプリケーションからVTDを追加します。
  引数:
    app_name(str): アプリケーション名
    user(int | Any): ユーザーID or Userオブジェクト(のはず)
    amount(int): 金額
```
